# 需求规格说明书

## 项目名称
剪贴板历史管理软件（Clipboard History Manager）

## 目标用户
Windows 电脑用户，日常办公中需要频繁复制粘贴文字和图片。

## 功能需求

### F1 - 剪贴板监控
- 软件后台运行时，自动检测系统剪贴板变化
- 支持文字内容（纯文本）的捕获
- 支持图片内容（位图）的捕获
- 每 500ms 轮询一次，去重（相同内容不重复记录）

### F2 - 历史记录存储
- 所有记录存储在本地 `E:\剪切板历史保存\`
- 文字内容存入 SQLite 数据库
- 图片以 PNG 文件存入 `E:\剪切板历史保存\images\`，数据库存路径

### F3 - 历史面板
- 以卡片列表形式展示所有历史记录
- 按时间降序排列（最新在上面）
- 置顶记录始终排在最上方
- 文字卡片：显示内容预览（截断至 80 字符）
- 图片卡片：显示缩略图
- 每条记录显示时间戳

### F4 - 再次粘贴
- 点击文字卡片：将该文字写入系统剪贴板
- 点击图片卡片：将该图片写入系统剪贴板
- 粘贴后面板自动隐藏

### F5 - 置顶功能
- 每条记录有「置顶/取消置顶」按钮
- 置顶后记录始终在最上方，带视觉标识

### F6 - 删除功能
- 每条记录有「删除」按钮
- 删除数据库记录，同时删除对应图片文件（如有）

### F7 - 搜索功能
- 搜索栏输入关键词
- 实时过滤文字内容包含关键词的记录
- 搜索结果仍保持时间降序 + 置顶优先

### F8 - 保留期限设置
- 设置面板提供三个选项：1天 / 3天 / 5天
- 默认保留 3 天
- 软件启动时 + 每小时自动清理过期记录

### F9 - 系统托盘
- 软件最小化到系统托盘，不占用任务栏
- 托盘图标右键菜单：打开历史 / 设置 / 退出
- 双击托盘图标：打开历史面板

### F10 - 全局快捷键
- 按下 `Alt+V` 弹出/隐藏历史面板
- 面板显示在鼠标所在屏幕的中央

## 非功能性需求
- 仅支持 Windows 平台
- 所有数据本地存储，不上传网络
- 界面语言：简体中文
- UI 风格：简洁直观，淡蓝色主色调
